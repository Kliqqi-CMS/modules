Created: 09/15/2009
Last Updated: 09/16/2009
Requires: Nothing


Installation

Upload the module and install it from the Admin > Modules page. If you are using URL Method 2 SEO URLs on your site you may need to add the following line to your .htaccess file. Pligg version 1.0.2 and above will already include this line in the htaccess file.

RewriteRule ^toolbar/(\d+)/?$ modules/pligg_web_toolbar/toolbar.php?id=$1


Description

This module will turn your external urls into Toolbar urls that will add a toolbar to the top of all external stories. This keeps users on your site as they read stories and gives them access to the vote count, comment count, story url, homepage and bookmarking tools.


Known Bugs and Future Features

  • Disabling the module from the Admin panel doesn't actually disable the toolbar. Please just disable it from the module management page.
  • Stories that do not have a URL (left blank or as http:// in submit step 1) result in an error popup.
  • A working vote button needs added so users can vote from the toolbar page
