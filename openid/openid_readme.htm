OpenId Login v0.1

Created by: AshDigg
Created on: 2007/04/14
Requires: Profile Extra Fields v0.2+
Requires: CURL Extension for PHP. php_curl.dll
Description
Allows people to login with OpenId. To use this mod. your usersmust first create a regular account on your site and then go into thier profile and enter thier OpenId url. They will then be able to login using OpenId in the future.
