LDAP Authentication v0.10

Description

Authenticates a user against an LDAP Server.

Installation

This module requires that you modify 1 core Pligg file. Please open /libs/login.php and find the line:

if ($user->user_id > 0 && $user->user_pass === $saltedpass && $user->user_lastlogin != "0000-00-00 00:00:00" && $user->user_enabled) {

Replace it with:

if ($user->user_id > 0 && $user->user_pass === $saltedpass && $user->user_enabled) {

This change removes the ability for Pligg to block users who have not validated their email address yet.
