RSS Importer Module

Description

Import RSS feeds from your favorite websites into your Pligg site.

Instructions

  1. Click on "Add a new feed."
  2. Enter feed URL in "Feed URL." input
  3. Click on the "Edit" link to configure the feed settings
  4. Click "Add a new field link" to map the fields of the RSS feed to your Pligg site fields. For example:
    • - link_title
    • - link_url
    • - link_content
  5. Set up a cron job to access the /modules/rss_import/import.php file to automatically fetch new stories.
