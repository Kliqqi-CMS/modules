SFS Antispam

Created by: ChuckRoast
Created on: 07-3-2010
Requires: None
Description
Helps determine if a user account is a spammer by connecting to the stopforumspam.com database.
If a user is on the global blocklist they will not be able to register.
Get a API key here
Add your api key in line 11 of modules/sfs_antispam/templates/sfs_antispam_index.tpl where it says APIKEYHERE
