Description

Basic Profanity Filter

Instructions

Copy 'outputfilter.profanity_filter.php' from the modules 'plugin' folder to your Pligg '/plugin' folder and then enable the module. To add or remove words, edit the 'outputfilter.profanity_filter.php' file after you copy it to the '/plugins' folder.
