GreyBox Javascript Lightbox

Created by: Yankidank
Created on: 07/16/2009

Includes

  • GreyBox v5.53

About

Greybox offers a javascript library independent solution for displaying lightbox popups. This means that you can use it in addition to having other javascript libraries installed like jQuery or Mootools without a major conflict that would break javascript features.

Usage

To trigger GreyBox you'll only need to append a rel tag on your <a> tags:

Example Upload Module Settings

If you use the Pligg Upload module you should set the Manual Thumbnail HTML format field to:
<a href="{target}"rel="gb_imageset[gallery]" ><img src="{path}" /></a>

Example 500x500 Popup Link

<a href="http://google.com/" title="Google" rel="gb_page_center[500, 500]"> Your Trigger Link Text Here </a>

Example Full Page Popup Link

<a href="http://google.com/" title="Google" rel="gb_page_fs[]"> Your Trigger Link Text Here </a>
