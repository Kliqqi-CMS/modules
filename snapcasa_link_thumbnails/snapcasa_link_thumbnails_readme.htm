SnapCasa Link Thumbnails

Created by: Eric "Yankidank" Heikkinen

Credit to: David Walsh

Created on: 07-01-2009

Requires: Mootools javascript library

Description

Inserts a SnapCasa.com thumbnail image when you hover over an external URL link on your website.

Requirements

This module uses MooTools and cannot be run in conjunction with a conflicting javascript library such as scriptaculous, prototype or jquery.

You must register an account with SnapCasa and tell it what domain you are using in order for this module to work. They allow up to 500,000 free snapshop impressions before asking you to upgrade to a paid plan. The free account is simple to set up and literally only takes 5 seconds to register for. Once you have registered you will need to edit the /modules/snapcasa_link_thumbnails/templates/thumbnails_head.tpl file. Line 26 requires you to enter in your own ID where you see ####.
