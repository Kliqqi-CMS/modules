Profile Extra Fields v0.2

Description

Allows users to specify a digg username in their profile. Rss feeds are then added for each digg user's digg submit and vote pages into a particular pligg category.
Make sure you update digg_users_settings.php with the right digg and pligg settings (aka category ids to store in, rss settings, etc)
