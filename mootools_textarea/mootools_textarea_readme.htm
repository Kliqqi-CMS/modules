Description

This module is based on the UvumiTools Textarea Mootools script. It will both add a counter below the submission and comment textareas and expand the height of the text area as more content is added to it.

Suggested Template Edits

These template edits are not required for this module to function, but they will remove the character count tool from Pligg's story submission process. The character count is no longer needed since this module basically replaces the feature with a count down, rather than the Pligg-based count up method.

Remove from your template's submit_step_2.tpl:

        function textCounter(field, countfield, maxlimit) {
                if (field.value.length > maxlimit) // if too long...trim it!
                        field.value = field.value.substring(0, maxlimit);
                        // otherwise, update 'characters left' counter
                else
                        countfield.value = maxlimit - field.value.length;
        }
function counter(text)
{
text.form.text_num.value = text.value.length;
}
Also remove this line found further down on submit_step_2.tpl:
<input readonly type=text name=remLen size=3 maxlength=3 value="400">{#PLIGG_Visual_Submit2_SummaryCharactersLeft#}
