Created by: Yankidank and Ashdigg
Updated on: 2008/11/04

Description

The Pligg Version Check Module writes two files to the module directory:
current.txt contains the version number stored in your database
latest.txt fetches the latest version number from Pligg.com.
It then compares the two files, and if it detects a difference an alert message appears on your admin panel homepage.
