Moo Login Module v0.1

Created by: Yankidank
Created on: 09/12/2008
Requires: {checkActionsTpl location="tpl_pligg_body_start"} and {checkActionsTpl location="tpl_pligg_head_end"} in pligg.tpl

Description

This module inserts an ajax tab to the top right of your site, when clicked on it drops down a login and search form using Mootools.
This module could potentially break your template. Test it on a local computer before activating on an active site to see if it works flawlessly.
